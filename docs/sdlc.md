# SDLC

SDLC means Software Development Life Cycle.

## Project Lifecycle

```text
Planning
-> Requirements
-> Design
-> Development
-> Testing
-> Deployment
-> Maintenance
```

## Applied To AI Mantra Generator

## Planning

Define whether the AI Mantra Generator solves a real user need.

## Requirements

User should be able to:

- Enter a prompt.
- Select keywords.
- Select voice.
- Select music.
- Select length.
- Use quick prompt examples.
- Start processing.
- Continue to Playground.

## Design

Translate Figma references into frontend components and interaction states.

## Development

Build the static prototype first.

Future integrations may include:

- Text generation.
- Text-to-speech.
- Music or soundscape generation.

## Testing

Verify:

- UI layout.
- Interaction states.
- Dropdown behavior.
- Navigation.
- Responsive breakpoints.

## Deployment

Prepare static hosting or app deployment after MVP prototype stabilizes.

## Maintenance

Iterate on:

- More voices.
- More music styles.
- Better prompt presets.
- Export and sharing.
